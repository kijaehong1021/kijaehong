- **저자**: Yifan Ao 외

## 문제 정의

GPU DB 시스템들의 실제 성능 병목이 어디에 있는지 체계적인 분석이 부족. 어떤 연산이 GPU에서 잘 되고, 어떤 연산이 안 되는지 명확하지 않음.

---

## 핵심 아이디어

GPU DB 시스템들을 다양한 워크로드로 프로파일링하여 성능 특성 분석:

- 주요 DB 연산(scan, join, aggregation, sort)별 GPU 활용도 측정
- 메모리 대역폭 vs 컴퓨팅 활용률 분석
- 병목 지점 식별 및 최적화 가이드라인 제시
